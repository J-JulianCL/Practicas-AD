# Práctica de Aplicaciones Distribuidas

Este proyecto implementa una serie de **servicios web con Node.js y Express** para la materia de Aplicaciones Distribuidas.

## Requisitos

- Node.js instalado
- Postman (para probar los servicios)

## Instalación

Ejecuta los siguientes comandos en la terminal:

npm init -y
npm install express

Para iniciar el servidor:
node index.js

Servidor:
http://localhost:3000

Leer el reporte para más detalles.
