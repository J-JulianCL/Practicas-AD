# Práctica 3 de Aplicaciones Distribuidas

Este proyecto implementa una práctica de repaso para la materia de Aplicaciones Distribuidas.

## Requisitos

- Node.js instalado
- Postman (para probar los servicios)

## Instalación

Ejecuta los siguientes comandos en la terminal:

npm init -y
npm install express

Para iniciar el servidor:
node index.js

Servidor:
http://localhost:3000
